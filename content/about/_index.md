# Đào Hữu Tiến

**Sinh viên Công nghệ Thông tin**  
Sinh năm 2004 | Đại học Công nghệ TP.HCM (HUTECH)

Xin chào! Mình là Đào Hữu Tiến, hiện đang là sinh viên chuyên ngành Công nghệ Thông tin tại **Đại học Công nghệ TP.HCM (HUTECH)**. Mình đam mê học lập trình, phát triển ứng dụng và luôn tìm kiếm cơ hội học hỏi các công nghệ mới.

![Avatar cá nhân](/images/avatar.jpg)

## Giới Thiệu

Mình bắt đầu hành trình lập trình từ năm nhất đại học và hiện đang tập trung phát triển kỹ năng **Full-stack Web Development**. Mình thích xây dựng các ứng dụng thực tế, giải quyết vấn đề và chia sẻ kiến thức qua blog cá nhân.

## Học Vấn

### Đại học Công nghệ TP.HCM (HUTECH)
- **Chuyên ngành**: Công nghệ Thông tin
- **Thời gian**: 2022 – Hiện tại (Dự kiến tốt nghiệp 2026)
- **GPA**: 3.25/4.0
- Các môn học nổi bật: Lập trình Web, Cơ sở dữ liệu, Lập trình ứng dụng di động, Mạng máy tính 

## Kỹ Năng

### Frontend
- HTML/CSS
- JavaScript (ES6+)

### Backend
- ASP.NET
- Spring Boot
- SQL Server 

### Công cụ & Khác
- Git & GitHub
- Postman

### Kỹ Năng Mềm
- Làm việc nhóm
- Giải quyết vấn đề
- Tiếng Anh B1
- Quản lý thời gian
- Thích ứng và học hỏi

## Dự Án Nổi Bật

### Website Bán sách (ASP.NET)
- Đồ án nhóm môn Phát triển ứng dụng Web
- Tính năng: Giỏ hàng, tìm kiếm sản phẩm, thanh toán giả lập
- Công nghệ: ASP.NET, SQL Server

### Blog Cá Nhân
- Trang chia sẻ kiến thức lập trình
- Công nghệ: Hugo, Markdown, Deploy trên Github
